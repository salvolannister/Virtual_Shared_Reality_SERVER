This repository contains the server for the application XListener created for my final thesis at the Polytechnique of Turin.  
It is based on AspNetCore version 2.2 and enables to create a solid base for the players to exchange informationa about the gameObjects 
and the virtual/augmented environment where they are immersed.
## Getting Started

1. Clone the project
2. Run the project
### Prerequisites
To run this project I used Visual Studio 2019, these are the steps that you should do:

1. Clone the repository
2. Install:
   - AspNEtCore.App v2.2.0 you can find it at https://dotnet.microsoft.com/download/dotnet-core/thank-you/sdk-2.2.100-windows-x64-installer
   - System.Text.Json
3. Add your IP in the appsettings.json 
4. Run selecting your configuration from the VS Editor
